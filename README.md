# 🇧🇷 Brasil Encantado

O Brasil Encantado é um projeto de site informativo e turístico desenvolvido para apresentar a diversidade cultural, as belezas naturais e a riqueza gastronômica de diversos estados brasileiros. O site oferece roteiros, dicas de especialistas e um canal direto de contato para planejamento de viagens.



## 🚀 Estrutura do Projeto

O site é composto pelas seguintes páginas:



* Início (index.html): Página principal.

* Destinos (destinos.html): Detalhamento de regiões.

* Gastronomia (gastronomia.html): Seção dedicada à culinária brasileira.

* Contato (contato.html): Formulário completo para contato.



## 🛠️ Tecnologias Utilizadas

* HTML5: Estruturação semântica de todas as páginas.

* CSS3: Estilização visual (arquivo brasil.css).

* Elementos Multimídia: Incorporação de vídeos do YouTube e galeria de imagens.



## 📝 Informações do Projeto

* Integrantes: Emanuele Pereira Pauli

* Ano de Desenvolvimento: 2026
